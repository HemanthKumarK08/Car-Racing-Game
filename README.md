Car Racing Game

Car Racing Game is a desktop-based 2D racing game developed using Java, Swing, and AWT. The objective of the game is to control the player's car, avoid incoming enemy vehicles, and survive as long as possible to achieve the highest score.

Features
Modern racing-themed user interface
Smooth 2D gameplay
Player-controlled car movement
Random enemy vehicle generation
Collision detection
Dynamic speed increase
Easy, Medium, and Hard difficulty levels
High-score management using file handling
Pause and Resume functionality
Game Over screen with restart option
Background music and crash sound effects
Modular Object-Oriented Programming (OOP) design
Technologies Used
Java
Java Swing
Java AWT
Object-Oriented Programming (OOP)
Graphics2D
Java Sound API
File Handling
Controls
Left Arrow / A → Move Left
Right Arrow / D → Move Right
P → Pause / Resume
R → Restart after Game Over
ESC → Return to Main Menu

This project was developed as part of an MCA Java project to demonstrate Java GUI programming, animation, event handling, collision detection, and game development concepts.
